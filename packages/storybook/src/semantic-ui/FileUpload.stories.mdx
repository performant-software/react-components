# Examples
#

### Size
Using the `maxSize` prop, file sizes and be limited to exclude files exceeding a certain size.

<Preview>
  <Story name='Size'>
    <FileUpload
      maxSize={10485760}
      onFilesAdded={action('files-added')}
    />
  </Story>
</Preview>

### Type
Using the `filesTypes` prop, files can be limited to a specific set of file extensions.

<Preview>
  <Story name='Types'>
    <FileUpload
      fileTypes={[
        'jpg',
        'jpeg',
        'png'
      ]}
      onFilesAdded={action('files-added')}
    />
  </Story>
</Preview>
