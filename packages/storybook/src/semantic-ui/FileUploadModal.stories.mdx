### Hide Button
The button to open the modal can be omitted by including the `includeButton` prop with a value of `false`.

<Preview>
  <Story name='NoButton'>
    <FileUploadModal
      button='Upload files'
      includeButton={false}
      itemComponent={({ item }) => (
        <Form.Input
          label='Name'
          value={item.name || ''}
        />
      )}
      onAddFile={(file) => file}
      onClose={action('close')}
      onSave={() => {
        action('save')();
        return Promise.resolve();
      }}
    />
  </Story>
</Preview>
